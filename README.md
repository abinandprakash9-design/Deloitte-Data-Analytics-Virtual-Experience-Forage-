# Factory Downtime Analysis Dashboard

## 📊 Project Overview
This project analyzes machine downtime across different factories and device types using Tableau and Excel.

## 🛠 Tools Used
- Tableau
- Microsoft Excel

## 📌 Key Insights
- Identified factories with highest downtime
- Compared downtime across device types
- Highlighted major causes of machine failure
